# 落笔 Nextype — Windows 端产品需求文档

## 产品定位与目标

落笔 Nextype Windows 端是从 Mac 端移植而来的桌面应用，保持与 Mac 端功能对齐，核心场景是「手机打字，电脑上屏」。应用以系统托盘常驻方式运行，通过托盘图标提供所有入口。

---

## 核心功能清单

### 1. 设备配对
- 配对码生成（4 位数字，7 种易记模式）
- 配对码有效期 60 秒
- 二维码配对
- 配对完成通知

### 2. 剪贴板同步
- 写入剪贴板
- 自动粘贴（Ctrl+V）
- 粘贴+回车
- 后缀追加
- AES-256-CBC 加密传输
- ACK 回执

### 3. 快捷键远程控制
- 发送（send）
- 插入（insert）
- 清空（clear）
- 模拟点击（tap）
- 模拟长按（longpress）
- 防抖机制 500ms

### 4. 快捷键录入
- 使用前端 JavaScript 实现
- 支持 Ctrl/Alt/Shift/Win 修饰键
- 不支持 Fn 键（Windows 限制）

### 5. 屏幕坐标配置
- 动态多维度配置
- 自动捕获新尺寸
- 三级容错匹配

### 6. 系统托盘
- 托盘菜单（连接手机、偏好设置、在线设备、日志、退出）
- 图标状态切换（未连接/已连接）
- 左键点击弹出菜单

### 7. 偏好设置窗口
- 基础设置（开机启动、清空剪贴板）
- 连接手机（配对码、设备列表）
- 使用数据（统计信息）
- 远程控制（快捷键配置）
- 关于和反馈

### 8. 引导流程
- 4 步引导（欢迎、权限、隐私、体验）
- Windows 不需要辅助功能权限，跳过权限页

### 9. 日志系统
- 四层架构（控制台、文件、内存、前端推送）
- 日志窗口（实时查看、导出、清空）

### 10. 使用统计
- 累计同步次数、字符数
- 今日统计
- 可禁用

### 11. 系统集成
- 开机启动（Windows 注册表）
- 系统托盘常驻
- 单实例保护
- 窗口关闭拦截

### 12. 中继连接
- WebSocket 中继
- 自动重连
- 心跳机制
- 信任列表同步

---

## 与 Mac 端的差异

| 功能 | Mac 端 | Windows 端 |
|------|--------|-----------|
| 剪贴板粘贴 | AppleScript (Cmd+V) | SendInput API (Ctrl+V) |
| 快捷键 | CGEventTap + global-shortcut | global-shortcut only |
| Fn 键支持 | 支持 | 不支持 |
| Dock 图标 | 可显示/隐藏 | 无此概念 |
| 辅助功能权限 | 需要 | 不需要 |
| 开机启动 | LaunchAgent | 注册表 |
| 托盘图标 | Template 图标 | .ico 图标 |
