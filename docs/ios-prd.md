# 落笔 Nextype — iOS 端产品需求文档

## 产品定位与目标

落笔 Nextype iOS 端是一款"手机输入，电脑出字"的跨设备文字同步工具的 iPhone/iPad 客户端。用户在 iOS 设备上通过键盘输入文字，内容通过公网中继实时同步到电脑端，支持插入（粘贴到光标位置）和发送（粘贴+回车）两种模式。

## iOS 平台特有限制说明

1. **无 AccessibilityService**：iOS 不提供类似 Android 的辅助功能服务 API，无法实现模拟点击功能
2. **后台执行受限**：iOS 对后台任务有严格限制，WebSocket 进入后台后被系统断开。已实现后台断连+前台重连策略
3. **屏幕亮度控制是全局设置**：`UIScreen.main.brightness` 影响全局，不像 Android 可以只控制当前窗口
4. **麦克风权限弹窗**：iOS 要求运行时请求麦克风权限

## 用户流程

### 首次使用流程
1. 打开应用 → 显示空状态引导页
2. 应用自动尝试从中继服务器恢复之前的配对信息
3. 如无可恢复的配对，用户点击"配对电脑"按钮
4. 输入电脑端显示的 4 位配对码
5. 通过中继服务器验证配对
6. 配对成功 → 进入主输入界面

### 日常使用流程
1. 打开应用 → 自动连接中继服务器并恢复上次设备（多设备时自动切换到最近活跃的电脑）
2. 在输入框中键入文字
3. 点击"插入"将内容同步到电脑光标位置
4. 点击"发送"将内容同步到电脑并附带回车
5. 30秒无操作屏幕自动变暗，触摸即唤醒
6. 进入后台自动断连，返回前台自动重连
