落笔 Nextype

隐私政策

最后更新：2026年3月

落笔 Nextype（以下简称"我们"或"本应用"）非常重视用户隐私。本隐私政策说明我们如何收集、使用和保护您的信息。

一、信息收集说明

1. 设备标识符

我们使用 Android 系统的 ANDROID_ID 生成稳定的设备标识符，用于与您配对的电脑端建立连接。该标识符在本地生成，不会直接上传至我们的服务器。

2. 麦克风权限

当您使用语音输入功能时，我们需要访问麦克风。录音数据将通过您自行配置的火山引擎 API 进行语音识别处理。我们不直接接收或存储您的音频数据。

3. 配对设备信息

配对时生成的设备 ID、设备名称和加密密钥均存储在您手机本地，采用加密存储方式保护。

4. 火山引擎 API 凭证

如您使用语音输入功能，需自行填写火山引擎 APP ID 和 Access Key。这些凭证存储在您手机本地，仅用于调用火山引擎语音识别服务，不会传输至我们的服务器。

5. 中继服务器通信

为实现手机与电脑的跨网络连接，您的设备 ID、设备名称等信息会通过我们的中继服务器（nextypeapi.yuanfengai.cn）进行路由。传输内容采用端对端加密，服务器无法解密您传输的内容。

二、信息安全

我们采用行业标准的加密措施保护您的数据：

• 配对加密密钥使用 AES-256 加密存储在您的设备本地

• 设备间通信采用端对端加密，内容在传输过程中无法被第三方解密

• 应用不存储、不传输与您个人身份直接相关的信息

三、第三方服务

语音识别功能使用火山引擎服务（由字节跳动提供）。使用该功能时，您的音频数据将传输至火山引擎服务器进行处理，请同时参阅火山引擎的隐私政策。

四、您的权利

您可以随时：

• 在应用内解除设备配对，清除配对数据

• 卸载应用，删除所有本地存储数据

• 拒绝同意本隐私政策，选择不使用本应用

五、联系我们

如有隐私相关问题，请访问 yuanfengai.cn 与我们联系。

本隐私政策自 2026年3月1日起生效。
